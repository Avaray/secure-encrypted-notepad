# SEN - Project Roadmap

This document tracks the development phases and release plans for the **SEN (Secure Encrypted Notepad)** project leading up to the version 1.0.0 release.

---

## 📍 Version 0.6.x - Foundation & Refinement (Current)
*Focus: Core building, aesthetics, and feature expansion.*

- [x] **UI/UX Overhaul**: Global vertical centering of elements (center_row).
- [x] **Theme Editor**: Granular customization of colors and geometry.
- [x] **Batch Converter**: Layout polish and toolbar integration.
- [ ] **Feature Polish**: Small quality-of-life improvements and immediate bug fixes.
- [ ] **Documentation**: Keeping the codebase well-documented as features grow.

## 📍 Version 0.7.0+ - Advanced Engineering
*Focus: Implementing the complex technical challenges from todo.md.*

- [ ] **Stealth Mode**: Binary noise format for plausible deniability.
- [ ] **Multicursor Support**: Advanced text editor synchronization.
- [ ] **Cargo Workspace**: Refactoring for multi-platform (Android) support.
- [ ] **System Icon Integration**: Dynamic icon sets for folders and files.

## 📍 Version 0.8.0+ - Stability & Localization
*Focus: Feature freeze and internationalization.*

- [ ] **Internationalization (I18n)**: Implementation of multi-language support.
    - Languages: 🇺🇸 **EN**, 🇩🇪 **DE**, 🇪🇸 **ES**, 🇫🇷 **FR**, 🇵🇱 **PL**.
- [ ] **Code Audit**: Final checks on encryption performance and UI responsiveness.
- [ ] **Stability**: No major architecture changes after this phase.

## 📍 Version 0.9.0+ - Public Testing & Optimization
*Focus: User feedback and final polish.*

- [ ] **Beta Testing**: Real-world usage testing by end-users.
- [ ] **Feedback Loop**: Fixing edge cases and UI quirks reported by testers.
- [ ] **Performance Tuning**: Optimizing decryption speed and memory usage.

## 📍 Version 1.0.0 - Official Release
*Goal: Production-ready secure notepad.*

- **📅 Target Date**: May 2026
- **Objective**: Stable, secure, and user-friendly release of SEN for all major platforms.
